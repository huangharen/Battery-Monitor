# 更新日志

## v1.1

### 新增功能

### Bug 修复

#### 电池数据
- 修复放电时功率乱码（`Rate` 字段无符号溢出）
- 修复充电时无法读取充满时间（`EstimatedTime` + 容量差自力计算）
- 修复功率中途变 0 不再恢复（取消满电时强制改写充电状态）
- 修复首次 Rate=0 时多次重试采样
- 修复历史变化率推算功率（终极不依赖驱动）
- 修复 psutil 返回 `0xFFFFFFFF` 堵塞所有 fallback

#### UI 与交互
- 修复浅色模式文字可读性（加深文字 + 提纯背景）
- 修复设置按钮同时选中（去掉 `setCheckable`，改用 `clicked` 手动管理）
- 修复拖拽卡顿（暂停动画定时器，松手恢复）
- 修复弹黑窗口（所有 `subprocess.run` 加 `CREATE_NO_WINDOW`）
- 修复插拔充电线闪退（全局 `try/except` 保护）
- 修复 `_draw_arc_glow` 缺少 dark 参数导致崩溃
- 修复 `QSoundEffect.load()` 方法不存在

#### 性能
- 渲染帧率 60fps → 30fps（CPU 占用减半）
- 移除 `powercfg /batteryreport` 阻塞调用（2-3s → WMI 100ms）
- 电源计划查询缓存，切换用 `Popen` 异步
- `moveEvent` 移除频繁磁盘写入，松手一次性保存

### 技术细节

- Python 3.12 + PyQt6 + psutil + wmi
- 数据源优先级：CallNtPowerInformation → psutil → 历史变化率线性回归
- 设计容量读取：WMI → powercfg /batteryreport
- 六层 QPainter 玻璃绘制：底色 / 阴影 / 色差 / 焦散 / 高光 / 镜面反射
- Win11 原生圆角：DWM API `DWMWA_WINDOW_CORNER_PREFERENCE`
- 窗口图标：charge.png
- 字体：HarmonyOS Sans SC Bold（全局 + 绘制通用）
